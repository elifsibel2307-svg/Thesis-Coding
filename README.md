## Overview

This repository contains all analysis code for my MSc thesis. The study investigates whether SBERT-derived textual embeddings extracted from startup descriptions improve the prediction of seed-to-Series A funding progression beyond traditional structured investment metrics.

Three classification algorithms are compared in a structured-only baseline condition and a text-enhanced condition: Logistic Regression, Random Forest, and XGBoost. SHAP-based interpretability analysis and exploratory subgroup analysis are included.
